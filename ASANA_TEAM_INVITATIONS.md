## Invitation Process

### Step 1: Prepare Invitation List
- [ ] Verify all email addresses are correct
- [ ] Confirm team structure is accurate
- [ ] Identify any missing team members
- [ ] Update this document with latest information

### Step 2: Send Invitations
1. Log into Asana as Admin
2. Navigate to "Add Members" in the main navigation
3. Enter email addresses from the list above
4. Assign appropriate roles:
   - Admin: Project/Team Managers
   - Member: Regular team members
   - Guest: Stakeholders and external collaborators
5. Add personal message (optional)
6. Click "Send Invitations"

### Step 3: Track Invitation Status
- [ ] Monitor invitation acceptance
- [ ] Follow up with non-responders after 24 hours
- [ ] Update status column in this document
- [ ] Resolve any delivery issues

### Step 4: Verify Access
- [ ] Confirm all team members can log in
- [ ] Verify correct roles are assigned
- [ ] Check project access permissions
- [ ] Test notification settings

## Role Descriptions

### Admin
- Full access to organization settings
- Can manage billing
- Can add/remove users
- Can configure integrations
- Can create/edit all projects

### Member
- Can create and manage their own tasks
- Can be assigned to tasks and projects
- Can comment and collaborate
- Can view projects they've been added to

### Guest
- Limited access to specific projects
- Can view and comment on tasks they're added to
- Cannot create projects or teams
- Cannot see other projects in the organization

## Troubleshooting

### Common Issues and Solutions

#### Issue: Invitation email not received
**Solutions**:
1. Check spam/junk folder
2. Verify email address is correct
3. Resend invitation from Asana
4. Contact IT to check email filtering

#### Issue: Can't accept invitation
**Solutions**:
1. Ensure using correct email address
2. Try different browser
3. Clear browser cache and cookies
4. Contact Asana support

#### Issue: Wrong role assigned
**Solutions**:
1. Admin can change role in "Admin Console" > "Users"
2. Remove user and re-invite with correct role
3. Contact support if issues persist

#### Issue: Can't access specific projects
**Solutions**:
1. Verify user was added to project
2. Check project privacy settings
3. Confirm user role permissions
4. Add user to project manually
